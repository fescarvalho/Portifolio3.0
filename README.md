## Steps to run the project

1. Clone the project

```
git clone https://github.com/fescarvalho/portifolio3.0
```

2. Navigate to project folder and Install Dependencies

```
cd portifolio3.0
npm install
```

7. Run the project in final version

```
npm run start
```
